# **운세 서비스 "명운(命運)" 웹앱 설계**

## **1. 개요**
- **프로젝트명:** 명운(命運) - 당신의 길을 비추는 지혜
- **목표:** 전통적인 운세 서비스를 현대적이고 세련된 웹 UI/UX로 재해석한 단일 페이지 웹앱(SPA) 개발. 사용자가 자신의 삶에 대한 통찰과 가이드를 얻을 수 있도록 돕는다.
- **역할:** 프로덕트 디자이너 + 프론트엔드 개발자

## **2. 화면 구조 및 플로우**
- **레이아웃:** 모바일 우선 반응형 디자인.
  - **Header:** 서비스 로고/이름 (`<h1>명운(命運)</h1>`)
  - **Navigation:** 4개의 탭 버튼 (`<nav>`). 각 탭 버튼은 `type="button"` 속성을 가지며, 텍스트와 함께 고유한 도형(정사각형, 원, 삼각형, 육각형)을 포함하여 시각적으로 구분.
  - **Main Content:** 탭 선택에 따라 동적으로 교체되는 콘텐츠 영역 (`<main>`). 각 탭의 상단에는 현재 탭에 대한 설명(`p#current-tab-description`)이 동적으로 표시.
  - **Footer:** 간단한 정보 (`<footer>`)
- **데이터 흐름:**
  1.  **사용자 입력:** 각 탭의 폼(생년월일, 관계 등)에 정보 입력.
  2.  **입력값 저장:** `localStorage`를 사용해 마지막 입력값을 브라우저에 저장하여 다음 방문 시 자동으로 불러옴.
  3.  **JavaScript 처리:** 입력값을 바탕으로 각 운세 로직(사주, 오늘의 운세 등)을 JS 함수 내에서 계산/생성. (MVP 단계에서는 복잡한 만세력 API 없이 자체 로직으로 간소화)
  4.  **결과 표시:** 처리된 결과를 받아 DOM 요소를 동적으로 생성하고, 카드 UI 형태로 화면에 렌더링.
  5.  **화면 전환:** 탭 클릭 시 다른 콘텐츠 섹션으로 부드럽게(Fade-in) 전환되며, 해당 탭의 도형과 설명 텍스트가 업데이트.

## **3. 탭별 기능 및 UI/UX 설계**

### **공통**
- **디자인:**
  - **색상:** 배경(Off-white: `#F8F7F2`), 기본 텍스트(Dark Navy: `#2C3E50`), 포인트/강조(Gold: `#D4AF37`, Beige: `#D2B48C`).
  - **폰트:** 가독성 높은 `Noto Sans KR` 또는 유사한 고딕 계열 웹폰트 사용.
- **UI:** 정보의 중요도에 따라 크기와 굵기를 조절한 타이포그래피. 직관적인 아이콘 사용 최소화, 텍스트 중심으로 정보 전달. 탭 선택 시 도형의 색상 및 크기 변화로 활성 상태 시각적 강조.
- **애니메이션:** 콘텐츠 전환 시 `0.5s`의 `fadeIn` 효과 적용. 도형 활성 시 `translateY` 및 `scale` 효과 적용.
- **톤앤매너:** "AI 분석", "100% 정확" 등의 문구 배제. "해석", "가이드", "경향" 등의 단어를 사용하여 사용자에게 안정감과 신뢰감을 줌.

### **탭 1: 사주팔자**
- **UI:** 생년월일, 시간, 성별 등 입력 폼 > 결과 버튼 > 로딩 후 결과 카드 표시.
  - 입력 필드 순서 조정: **이름** 아래에 **성별 선택**, **생년월일** 아래에 **양력/음력 선택**이 배치되도록 변경.
  - 출생시간 선택 드롭다운에 모든 전통적인 한국 시간대(자시부터 해시까지) 옵션이 정확한 시간 범위 및 자시 중복 없이 수정 및 추가됨.
- **결과 카드:**
  1.  **요약:** 사주팔자 8글자 표(시주 모를 시 '미상' 처리).
  2.  **디테일:** 오행 분포도(간단한 막대그래프), 타고난 성향/강점/주의점 텍스트 설명.
- **로직:** 간지(干支) 배열을 기반으로 생년을 이용해 단순 계산. 월주, 일주, 시주는 MVP에서 그럴듯한 값으로 생성하여 '참고용'임을 명시.

### **탭 2: 오늘의 운세**
- **UI:** 생년월일/별자리(선택) 입력 또는 "랜덤 생성" 버튼 > 결과 카드 표시.
- **결과 카드:**
  1.  **요약:** 오늘의 한줄 운세.
  2.  **디테일:** 금전/일&학업/건강/대인 관계 운세 아이콘과 함께 표시. 행운의 색/숫자/피해야 할 것 목록.
- **로직:** `new Date()`를 기반으로 한 시드(seed) 값을 사용, 하루 동안 동일한 랜덤 결과가 나오도록 구현.

### **탭 3: 점성술**
- **UI:** 생년월일 입력 폼 > 결과 카드 표시.
- **결과 카드:**
  1.  **요약:** 계산된 태양 별자리와 상징 이미지.
  2.  **디테일:** 별자리의 핵심 키워드 5개(칩 형태), 3~5 문장의 성향 해설.
- **로직:** 입력된 월/일을 기준으로 별자리를 판단하는 간단한 조건문 함수 구현.

### **탭 4: 연애운**
- **UI:** 본인/상대 생년월일, 관계 상태 선택 폼 > 결과 카드 표시.
- **결과 카드:**
  1.  **요약:** 오늘의 연애운 한줄 평.
  2.  **디테일:** 3가지 행동 조언. 상대 정보 입력 시 '궁합 점수'와 그에 대한 해석 문장 제공.
- **로직:** 궁합 점수는 두 생년월일의 타임스탬프를 조합한 단순 계산식으로 생성. 점수에 따라 다른 해석 문구를 보여줌.

## **4. 기술 스택 및 구현**
- **Frontend:** HTML, CSS, JavaScript (ES6+)
- **라이브러리:** 없음 (Vanilla JS)
- **배포:** 정적 파일(HTML/CSS/JS)을 웹 서버에 올리거나 로컬에서 직접 파일 열기.
